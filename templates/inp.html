Text Summarizer

Enter URL to a news article

How does it work?

This is a news article summarizer that takes the link of a news article, does some really simple yet complicated math, and returns a summary of what the article is talking about

It uses an algorithm called Text Rank - which is a modified version of Page Rank. Page Rank is an algorithm used in search engines such as Google to link and rank webpages for a particular search.

Steps Taken

1. Using Python and the BeautifulSoup module, I was able to extract the links from the RSS feeds of different newspaper franchises.

2. After extracting the article content, I stored the content and title in two seperate lists.

3. I ran the contents of the article through the TextRank algorithm. The algorithm splits the different sentences and vectorizes them.

4. Text Rank generates a similarity score for all the vectors and picks the sectences with top 5 scores.

5. To save the summaries and links, I used SQLAlchemy to construct a database.

I used Python and Flask to create these webppages and links between them!
